# Logo Design - Raleigh Home Automation Co.

## Logo Concept

The logo features a stylized oak tree (Raleigh's symbol) with a home icon integrated into the tree structure, representing the fusion of nature (Raleigh) and home automation technology.

### Design Elements
- **Oak Tree**: Stylized silhouette with layered branches
- **Home Icon**: Orange accent color, integrated into the tree trunk area
- **Accent Dots**: Small orange circles representing connectivity/automation nodes
- **Color Scheme**: 
  - Tree: #2C3E50 (Primary Gray)
  - Home/Accents: #FF6B35 (Bright Orange)
  - Background: Transparent/White

### Logo Files
- `logo.svg` - Square version (200x200px) - ideal for favicons, social media
- `logo-horizontal.svg` - Horizontal version with company name - ideal for headers

### Usage Guidelines
- Use on white or light gray backgrounds
- Minimum size: 50px for square logo, 200px width for horizontal
- Maintain aspect ratio when scaling
- Orange accent should always be visible for brand recognition

### Color Palette Reference
- Primary Gray: #2C3E50
- Light Gray: #ECF0F1
- White: #FFFFFF
- Accent Orange: #FF6B35
- Secondary Gray: #7F8C8D
